# 简易成绩计算系统 - GitHub 仓库简介
## 项目介绍
简易成绩计算系统是一款专为中国初高中学校教育管理设计的成绩分析工具，能够快速统计班级和年级的考试成绩，自动计算平均分、及格率、优秀率，并生成学生排名和班级评比结果。系统采用现代化的 PyQt5 图形界面，操作简便直观，支持多种数据导入和导出格式。

## 功能特点
### 核心功能
- 成绩数据导入 ：支持批量导入 Excel 成绩文件，自动识别班级、姓名、座号及各科成绩
- 一分三率计算 ：自动计算班级平均分、及格率、优秀率，支持自定义分数线
- 学生排名分析 ：生成学生总分排名、单科排名，支持多种排名规则（min/dense/average）
- 进退步计算 ：通过导入历史总表，计算学生成绩进退步情况
- 班级评比 ：生成班级汇总表，包含总分分数段统计和班主任信息
- 单科分析 ：提供单科班级分析和排名，包含比差和名次统计
- 教师配置 ：支持设置各班级班主任和各科任课教师，可导入/导出教师配置
- 报表导出 ：生成格式化的 Excel 结果报表，自动应用专业配色和格式

### 基本操作流程
1. 导入成绩数据
   
   - 点击工具栏"导入文件"按钮
   - 选择包含成绩的 Excel 文件（可多选，建议一个班一个成绩表，导入多个表格）
   - 系统自动识别数据结构
2. 设置参数
   
   - 在左侧面板设置排名规则和缺考处理方式
   - 在"科目与教师设置"选项卡中配置各科满分、及格线、优秀线
   - 系统会根据科目名自动填充常用默认值
3. 配置教师信息
   
   - 在"科目与教师设置"选项卡中设置各班级班主任和任课教师
   - 可通过"导入教师配置"按钮批量导入教师信息
4. 执行计算
   
   - 点击工具栏"计算"按钮
   - 系统自动计算并生成各种分析报表
5. 查看结果
   
   - 在不同选项卡中查看学生排名表、班级汇总表、单科分析等结果
   - 选择科目可查看详细的单科班级分析和排名
6. 导出报表
   
   - 点击工具栏"导出结果"按钮
   - 选择保存位置，系统生成包含多个工作表的 Excel 报表
### 进阶功能
- 进退步计算 ：导入历史总表并勾选"启用进退步计算"，系统会自动计算学生成绩进退步
- 自定义权重 ：可设置各科成绩在总分计算中的权重
